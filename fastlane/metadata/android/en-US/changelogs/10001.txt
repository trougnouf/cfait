## [1.0.1] - 2026-05-14

### 🚀 Features

- *(parser)* Support start-due time range (e.g. ^@2026-05-06 12:20-16:20)
- *(linux)* Warn users using the volatile keyutils keystore ( https://github.com/trougnouf/cfait/issues/29 )
- *(gui)* Unsynced icon tooltip shows unsynced items
- Optimize companion event sync using CalDAV REPORT queries and task UID metadata

### 🐛 Bug Fixes

- Decouple keyring access and ensure Linux Secret Service unlocking to fix password loss ( https://github.com/trougnouf/cfait/issues/29 )
- *(gui)* Restore broken GUI task list items and navigation
- *(linux)* Handle locked keyrings and prevent redundant config save ( https://github.com/trougnouf/cfait/issues/29 )
- *(flatpak-gui)* Resolve flatpak hang and prevent mid-session password loss
- *(gui)* Silence calloop log noise by replacing blocking sleep with tokio sleep
- Decouple disk persistence from network sync to resolve timeouts and 412 ETag conflicts ( https://github.com/trougnouf/cfait/issues/25 )
- *(gui)* Use in-memory state for UI refreshes to fix config race condition
- *(tui)* Restore and localize About/Support help screen
- *(android)* Fix Android filter flash by using local UI state after task dispatch
